## 대규모 스터디
-----
### 스터디 규칙
* 스터디는 매주 토요일 10시에 한다
* 요일 변경이나 시간대 변경은 정규 일정 3시간 전에 말한다
* 스터디는 오프라인을 지향하며 장소 예약은 돌아가며 진행한다
* 스터디 자료는 금요일 자정 전까지 제출하며 형식은 노션으로 정리한다
